## Description

CPP Module 00 is the first C++ module of the 42 curriculum. It introduces the basics of compiling C++ code with the correct flags, working with classes, and following the C++98 standard. The goal is to get comfortable with the language, the build workflow, and the exercises' structure.

### Exercises

- **ex00 - Megaphone**: write a program that transforms its arguments to uppercase.
- **ex01 - PhoneBook**: build a small contact manager using classes and formatted output.

### Compiling each exercise

Go inside the exercise folder and run:

```bash
make
```

To rebuild from scratch, use:

```bash
make re
```

### Cleaning files

To remove object files and the executable:

```bash
make clean
make fclean
```

## Submission

Submit the completed exercises from this module following the 42 requirements. Each exercise should compile with `-Wall -Wextra -Werror -std=c++98`.
