## Description

CPP Module 01 is the second C++ module of the 42 curriculum. It focuses on memory allocation, pointers, references, switch statements, and basic object-oriented programming. The goal is to become more comfortable with how C++ handles data, objects, and file manipulation.

### Exercises

- **ex00 - BraiiiiiiinnnzzzZ**: create and use a simple Zombie class.
- **ex01 - Moar Brainz!**: generate a horde of zombies dynamically.
- **ex02 - HI THIS IS BRAIN**: explore pointers and references.
- **ex03 - Unnecessary violence**: manage weapons and human classes.
- **ex04 - Sed is for losers**: replace text inside a file.
- **ex05 - Harl 2.0**: implement a complaint system with different log levels.

### Compiling each exercise

Go inside the exercise folder and run:

```bash
make
```

To rebuild from scratch, use:

```bash
make re
```

### Cleaning files

To remove object files and the executable:

```bash
make clean
make fclean
```

## Submission

Submit the completed exercises from this module following the 42 requirements. Each exercise should compile with `-Wall -Wextra -Werror -std=c++98`.
